## Build

1. [Install rust](https://www.rust-lang.org/tools/install).
2. Clone the repository and enter into it.
```bash
git clone https://github.com/davudMagomedov/popusk.git && cd popusk
```
3. Build the project.
```bash
cargo build --release
```
4. Binary in `target/release/` directory.
